# Banking API
[![License: MIT](https://img.shields.io/badge/license-GNU-fef3c0?labelColor=1b1b1f)](./LICENSE)


Banking API is a tool to keep track of money for small internal purposes. Admin accounts can be create user accounts who can share money. 
## How to set up
- run `npm install`
- run `npm start` or `npm dev` for live reload
